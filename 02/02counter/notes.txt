// React Hooks Overview

// 1. Introduction to React Hooks
// React hooks are functions that let you "hook into" React state and lifecycle features in function components.
// You can either use the built-in Hooks or combine them to build your own.

// 2. Benefits of Using React Hooks
// - Simplify code by eliminating the need for classes
// - Improve code readability and maintainability
// - Enhance performance by reducing unnecessary re-renders

// 3. Commonly Used React Hooks
// - useState: Manage state in functional components
// - useEffect: Handle side effects, such as API calls or setting timers
// - useContext: Access context (shared state) in functional components
// - useReducer: Manage state with a reducer function
// - useCallback: Memoize functions to prevent unnecessary re-renders
// - useMemo: Memoize values to prevent unnecessary re-rendersReact Hooks -
